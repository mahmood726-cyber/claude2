# Contributing to the R² Validation Project

We welcome contributions to this project! Whether you are reporting a bug, suggesting an enhancement, or submitting a pull request, your input is valued.

## How to Contribute

1. **Reporting Bugs**: Please use the GitHub issue tracker to report any bugs. Provide as much detail as possible, including steps to reproduce the issue.
2. **Suggesting Enhancements**: If you have ideas for new features or improvements, feel free to open an issue for discussion.
3. **Pull Requests**:
    - Fork the repository.
    - Create a new branch for your feature or fix.
    - Ensure your code follows the existing style.
    - Submit a pull request with a clear description of your changes.
